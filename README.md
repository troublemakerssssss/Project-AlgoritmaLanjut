Program ini adalah sistem manajemen toko game & CD berbasis konsol yang memungkinkan pengguna untuk menampilkan seluruh koleksi game beserta informasi judul, platform, tahun rilis, dan stok; menambahkan data game baru secara manual; mencari game berdasarkan kata kunci judul menggunakan Linear Search atau berdasarkan ID menggunakan Binary Search; serta mengurutkan daftar game berdasarkan tahun rilis (Bubble Sort) maupun judul secara alfabetis A–Z (Selection Sort), dengan manajemen memori dinamis menggunakan pointer.
